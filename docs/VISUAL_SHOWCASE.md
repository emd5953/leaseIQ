## üé≠ Animation Patterns

### Hover Effects
- **Cards**: Lift up 2px, shadow intensifies (duration-500)
- **Images**: Scale to 105% (duration-700)
- **Buttons**: Opacity to 90% (duration-300)
- **Links**: Color change to sage green (duration-300)

### Transitions
- **Page loads**: Fade in with slight upward movement
- **Modals**: Slide in from top with backdrop fade
- **Accordions**: Smooth height transition
- **Carousels**: Slide with ease-out curve

### Micro-interactions
- **Focus states**: Sage green ring with 2px width
- **Loading states**: Pulse animation on skeleton screens
- **Success states**: Check icon with scale animation
- **Error states**: Shake animation with red tint

## üì± Responsive Behavior

### Navigation
- **Desktop**: Horizontal nav with all links visible
- **Mobile**: Hamburger menu with full-screen overlay

### Grids
- **Desktop**: 3-4 columns
- **Tablet**: 2 columns
- **Mobile**: 1 column

### Typography
- **Desktop**: text-8xl headlines
- **Tablet**: text-6xl headlines
- **Mobile**: text-5xl headlines

### Spacing
- **Desktop**: py-32, gap-16
- **Tablet**: py-24, gap-12
- **Mobile**: py-16, gap-8

### Images
- **Desktop**: 16:9 or square aspect ratios
- **Mobile**: 3:4 aspect ratio to prevent excessive height

## üåü Signature Design Elements

### 1. Paper Grain Texture
- SVG fractal noise overlay
- 1.5% opacity
- Fixed position, covers entire viewport
- Creates warm, tactile feel

### 2. Arch-Shaped Images
- 200px border radius on top corners
- Iconic architectural moment
- Used for hero images and featured content

### 3. Staggered Cards
- Every 2nd card translated vertically
- Creates natural, organic flow
- Breaks rigid grid structure

### 4. Italic Emphasis
- Single words in headlines italicized
- Adds personal, handwritten touch
- Playfair Display italic variant

### 5. Soft Shadows
- Diffused, natural lighting
- Low opacity (5-15%)
- Multiple levels for depth

### 6. Pill-Shaped Buttons
- Fully rounded (rounded-full)
- Uppercase tracking-widest text
- Smooth hover transitions

### 7. Icon Circles
- Sage green background at 10% opacity
- Icons at 1.5 stroke width
- Used for features and stats

### 8. Generous Whitespace
- py-32 between sections
- gap-16 in grids
- Breathing room everywhere

## üéØ Design Principles Applied

1. **Organic Softness**: No hard angles, everything flows
2. **Typographic Elegance**: Large, airy headlines with high contrast
3. **Earthbound Palette**: All colors derived from nature
4. **Tactile Texture**: Paper grain makes it feel real
5. **Breathing Space**: Generous whitespace throughout
6. **Intentional Movement**: Slow, graceful animations
7. **Staggered Rhythm**: Breaking the grid for natural flow
